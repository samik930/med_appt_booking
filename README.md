# Medical Appointment Booking System

A modern, full-stack web application that connects patients with healthcare providers for seamless appointment scheduling.

### For Patients
- **User Authentication**: Secure signup and login
- **Doctor Search**: Find doctors by name or specialization
- **Appointment Booking**: Book available time slots
- **Appointment Management**: View, track, and manage appointments
- **Responsive Design**: Works on all devices

### For Doctors
- **Profile Management**: Manage personal and professional details
- **Availability**: Set available time slots
- **Appointment Management**: View and manage patient appointments
- **Patient Records**: Access patient appointment history
